# Add Gravitee CircleCI

This skill adds the `.circleci/config.yml` file configured to use Gravitee's standard CI/CD pipelines.

## Instructions

1.  **Create Directory**:

    ```bash
    mkdir -p .circleci
    ```

2.  **Add config.yml**:
    Create `.circleci/config.yml` using the template at `resources/config.yml.template`.
