# Project Decisions

This directory stores major project decision records for `APIM-10345 Member Mapping to Application`.

Decision records in this directory should use Epic-scoped identifiers and filenames such as:
- `APIM-10345-decision-01`
- `APIM-10345-decision-01-<short-slug>.md`

Use it for decisions where:
- multiple meaningful options were considered
- the trade-offs were discussed by the responsible group
- the chosen direction affects the Epic scope, UX, domain behavior, implementation path, or rollout

Each decision record should explain:
- the Epic-scoped decision identifier
- the decision topic
- the options considered
- the arguments for and against each option
- the people involved
- the agreed direction

The Epic index of these records lives in `../APIM-10345-requirements.md` under `## Project Decisions`.
