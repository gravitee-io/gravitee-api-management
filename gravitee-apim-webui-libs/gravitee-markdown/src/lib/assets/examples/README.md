# GMD example pages

- **`gmd-full-showcase.md`** — Single page that exercises Markdown, HTML, `gmd-grid` / `gmd-cell`, `gmd-card` (+ title / subtitle + themed colors), `gmd-button` (filled / outlined / text, internal and external links), fenced code blocks, tables, `details`/`summary`, and all form components: `gmd-input`, `gmd-textarea`, `gmd-select`, `gmd-radio`, `gmd-checkbox`, `gmd-checkbox-group`.

Paste the file body into a portal documentation page of type **Gravitee Markdown**, or open the **Gravitee Markdown Viewer → Full GMD showcase** story in Storybook (served from `assets/examples/`).
