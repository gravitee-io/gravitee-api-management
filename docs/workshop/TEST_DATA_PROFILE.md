### Total Requests card
- **Expected value:** ~130
- **Source:** `GET /analytics?type=COUNT`
- **Notes:** Exact count depends on how many requests errored due to network/gateway issues

### Avg GW Response Time card
- **Expected value:** 100–400 ms (varies by your network latency to httpbin.org)
- **Source:** `GET /analytics?type=STATS&field=gateway-response-time-ms`
- **Notes:** httpbin.org is a public service; latency depends on your location and network

### Avg Upstream Response Time card
- **Expected value:** Similar to GW response time (small overhead from Gravitee itself)
- **Source:** `GET /analytics?type=STATS&field=endpoint-response-time-ms`
- **Notes:** GW time ≥ upstream time; the difference is Gravitee overhead

### Avg Content Length card
- **Expected value:** Weighted average across batch sizes
  - Batch 2 (×15): ~80 B each
  - Batch 3 (×10): ~460 B each
  - Batch 4 (×5): ~4,900 B each
  - All other batches: 0 B (no body)
  - **Weighted avg ≈ (15×80 + 10×460 + 5×4900) / 130 ≈ 230 B**
- **Source:** `GET /analytics?type=STATS&field=request-content-length`
- **Notes:** Requests without a body contribute 0 to the average; the actual average shown
  will be lower than the POST-only average because 100 of 130 requests have no body

## Potential Issues

| Symptom | Likely cause | Fix |
|---------|-------------|-----|
| All cards show `—` | ES has no data yet | Wait 30 more seconds; check `curl localhost:9200/gravitee-v4-metrics-*/_count` |
| Total Requests is 0 but ES has docs | Analytics disabled on the API | Re-run script; it sets `analytics.enabled: true` on creation. Existing APIs need manual toggle in Console → Settings |
| Status pie shows only `200.0-300.0` | httpbin.org path not being stripped correctly | Verify gateway strips context path: `curl -v http://localhost:8082/workshop/status/404` should return 404, not 200 |
| Avg Content Length is 0 | POST requests not reaching httpbin | Check if POST /workshop/post returns 200: `curl -X POST http://localhost:8082/workshop/post -d '{"test":1}'` |
| All requests timeout (COUNT_ERR > 50) | httpbin.org rate-limiting or unreachable | Try `curl https://httpbin.org/get` — if slow, wait and retry |
| `401 Unauthorized` from Management API | Wrong admin credentials | Set `ADMIN_PASS=yourpassword ./scripts/generate-test-data.sh` |
| `jq: command not found` | jq not installed | `brew install jq` |
