General

Health-check
Enable health-check
By enabling health-check, the endpoint can be disabled automatically if expectations are not satisfied.
Backup endpoint
A "backup" endpoint is not included in the load-balancing algorithm, meaning that an incoming HTTP request is never transmitted to this kind of endpoint.

HTTP Configuration
Enable keep-alive
Use an HTTP persistent connection to send and receive multiple HTTP requests / responses.
HTTP pipe-lining
When pipe-lining is enabled requests will be written to connections without waiting for previous responses to return.
Enables compression (gzip, deflate)
The gateway can let the remote http server know that it supports compression, and will be able to handle compressed response bodies.
